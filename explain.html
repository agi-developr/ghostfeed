Architecture

Five agents, one pipeline, zero humans.
Phase 1

Trend Research

The Trend Scout agent analyzes the niche, identifies trending topics, and selects a unique angle that maximizes engagement potential.

GPT-4o / Claude
->
Phase 2

Script Writing

The Scriptwriter crafts a hook-first, multi-segment script with a strong opening, emotional arc, and call to action. Optimized for short-form retention.

GPT-4o / Claude
->
Phase 3

Visual Generation

The Visual Director generates scene-by-scene images matching the script segments. Each image follows the niche aesthetic in 9:16 vertical format.

DALL-E 3 / gpt-image-1
->
Phase 4

Voice Synthesis

Voice Forge synthesizes narration from the script using text-to-speech. The voice matches the niche tone — calm for stoicism, energetic for productivity.

OpenAI TTS / ElevenLabs
->
Phase 5

Video Assembly

The Editor combines images, audio, and burned-in captions into a final MP4. Handles transitions, timing sync, and caption styling via FFmpeg.

FFmpeg + ASS subtitles
The Five Agents

trend-scout

Research Agent

Scans trending topics within the configured niche. Evaluates engagement potential, identifies unique angles, and avoids overused topics from content memory.

in: niche config, content memory
out: { topic, angle, reasoning }

scriptwriter

Content Agent

Writes hook-first scripts optimized for short-form retention. Structures content into timed segments with visual directions for each scene.

in: topic, angle, niche tone
out: { hook, segments[], cta, title }

visual-director

Creative Agent

Generates image prompts for each script segment, then produces images via DALL-E 3. Maintains visual consistency across scenes using the niche aesthetic.

in: script segments, aesthetic
out: segment_0.webp ... segment_N.webp

voice-forge

Audio Agent

Converts the full script narration into spoken audio using text-to-speech. Selects voice profiles that match the niche tone and pacing.

in: script text, voice config
out: voiceover.mp3

editor

Assembly Agent

Orchestrates the final video: syncs images to audio timing, generates ASS subtitle files, burns captions into the video, and exports the final MP4.

in: images, audio, script
out: ghostfeed_XXXXX.mp4
